
Application created by [ThinkJS](http://www.thinkjs.org)

## Install dependencies

```
npm install
```

## Start server

```
npm run start
```

## Compile server

```
npm run compile
```

## Test with pm2

Use pm2 to deploy app on production enviroment.

```
pm2 startOrReload pm2_test.json
```

## Deploy with pm2

Use pm2 to deploy app on production enviroment.

```
pm2 startOrReload pm2_production.json
```
